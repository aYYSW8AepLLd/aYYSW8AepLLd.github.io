# 教程

## Python

### 第一课 1 安装 Python
发布日期：2023/04/21（YYYY/MM/DD） 20：25 星期五

请在 https://www.python.org/downloads/ 下载Python。

选择你想要的版本（这里使用 python3.7.5。如果是Windows XP系统，选择3.4.3（可能后续出现故障））
  
如果是macos系统，请选择下面的“Macintosh”。
  
接下来，查看你的系统位数，在控制面板\系统\系统与安全，如果是64位，则下载x64版本，如果是32版本，则下载x86版本（此方法仅适用于Windows）
  
如果是MacOS系统，我不知道，请在互联网上寻找解决方案。
  
下载完成后，点击安装包，勾选“Add Python to PATH”，点击“立即安装”按钮，然后一路点击“下一步”按钮，等待进度条运行满，安装完成。
  
按 Win + R 调用正在运行的框。在运行框中输入 cmd，在 cmd 中输入 py，如果没有错误，则安装成功并添加到环境变量中。如果你犯了错误，我建议你重新安装它。（此方法仅适用于 Windows）
